Now you can track your [@opencode](https://twitter.com/opencode) and [@claudeai](https://twitter.com/claudeai) CLI coding sessions in one place.

[opensync.dev](http://opensync.dev) provides searchable history, markdown export, and eval-ready datasets. 

See tool usage, token spend, and session activity across projects.
