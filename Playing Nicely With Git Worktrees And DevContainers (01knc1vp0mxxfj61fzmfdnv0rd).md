## The problem


From my `playground` repository, I created a worktree called `playground-wt-test` initialized to the `main` branch.



```
git worktree add ../playground-wt-test main  

```

I then opened the `playground-wt-test`folder in VSCode, and selected **Rebuild and Reopen in Container**. This resulted in the container being built successfully, the VSCode opening in `/workspaces/playground-wt-test`, but no detected git repository:


![](https://miro.medium.com/v2/resize:fit:628/1*I1M_9Q30D_WfBPiBlBJdVw.png)fatal: not a git repository: /.../playground/.git/worktrees/playground
