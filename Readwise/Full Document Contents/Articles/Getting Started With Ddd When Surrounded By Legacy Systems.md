- Summary: When working with legacy systems, create small "bubble" contexts that isolate new domain models using an Anticorruption Layer (ACL).  
Start with a modest, valuable feature so the team can experiment without changing the legacy system.  
If the bubble grows, evolve its model and ACL carefully or move to an autonomous context with synchronized data.
